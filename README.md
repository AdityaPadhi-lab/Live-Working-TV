# 📺 Live Working TV — Web IPTV Player

A **full-stack web IPTV player** that streams live TV channels (HLS `.m3u8`) using a custom backend and modern frontend.  
Play TV on **mobile**, **tablet**, or **desktop browser** with a searchable channel list.

🎯 Built With:
- **Node.js + Express** backend
- **React.js** frontend
- **HLS.js** for adaptive live stream playback
- Dynamic playlist parsing from public IPTV sources
- Optional advanced HLS proxy rewriting

---

## 📌 Features

✅ Fetches live TV streams from public `.m3u` playlists  
✅ Searchable channel list  
✅ Works on Browser, Android, iOS, Tablet  
✅ Handles CORS via proxy for HLS playback  
✅ Progressive loading to improve performance  
✅ Clean Channel Grouping & Logos

🧠 *Under the hood:*  
Backend parses M3U playlists → frontend loads channels → user clicks → HLS.js plays stream.
